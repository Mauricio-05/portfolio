# 🌟 Portfolio Personal

¡Bienvenido a mi portafolio personal! Este proyecto está construido con Next.js, React y TailwindCSS, y aquí muestro mis proyectos, habilidades y experiencia profesional de manera interactiva y moderna.

## 🚀 Características principales

- Animaciones suaves y efectos visuales llamativos
- Sección de proyectos destacados
- Información sobre mí y contacto
- Diseño responsivo y accesible
- Efectos interactivos (3D, seguimiento de ojos, etc.)

## 📂 Estructura del proyecto

- `components/` – Componentes reutilizables y UI
- `app/` – Páginas principales y layout
- `public/` – Imágenes y recursos estáticos
- `styles/` – Estilos globales
- `hooks/` – Custom hooks
- `lib/` – Utilidades y animaciones

## 🛠️ Instalación y uso

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Mauricio-05/portfolio.git
   cd tu-portafolio
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Abre [http://localhost:3000](http://localhost:3000) para ver tu portafolio.
